Amazon Pay PHP SDK Samples

  • One-Time Payment Sample - Standard checkout workflow for physical goods
  • Recurring Payment Sample - Enable subscription payments and other recurring scenarios
  • Callback Sample - Use Login Button Widget and Address Book/Wallet Widgets on the same page

In order for the samples to work properly, you must enter your MWS keys, merchant ID, and other settings into the config.php file.

The samples are designed to use the SDK from the amazon-pay.phar PHP archive file from the PHP SDK.
